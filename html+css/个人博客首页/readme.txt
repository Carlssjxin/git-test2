# 个人博客首页（HTML + CSS）

这是一个示例个人博客首页页面，包含文章列表和侧边栏。

文件说明：
- index.html ：网页结构文件
- style.css  ：样式文件
- README.txt ：项目说明文件

页面结构：
1. 顶部标题区（Header）
   - 博客名称、简介文字
2. 主体内容区（Main）
   - 左侧文章列表（标题、时间、摘要）
   - 右侧侧边栏（作者信息、分类、联系方式）
3. 页脚（Footer）
   - 版权信息

功能特点：
- Flex 双栏布局，左主右辅
- 卡片阴影与 hover 动画效果
- 响应式支持（≤768px 自动改为单列）
- 纯 HTML + CSS 实现，无 JS

使用方法：
1. 将三个文件放在同一目录下。
2. 在项目目录创建 images 文件夹，放入一张头像图片：avatar.jpg。
3. 打开 index.html，即可在浏览器查看效果。
