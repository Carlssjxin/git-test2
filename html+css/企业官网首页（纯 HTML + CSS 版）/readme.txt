# 企业官网首页（HTML + CSS）

本项目演示一个简洁的企业官网首页布局，纯 HTML + CSS 实现。

文件说明：
- index.html ：网页结构
- style.css  ：样式文件
- README.txt ：说明文件

页面模块结构：
1. 顶部导航栏（Logo + 菜单）
2. Banner 横幅（背景图 + 标题文字）
3. 公司简介（居中文本介绍）
4. 产品展示（3个产品卡片）
5. 页脚（版权 + 联系方式）

功能特点：
- Flex + Grid 混合布局实现响应与对齐
- 所有文字居中、卡片带阴影与悬停效果
- 结构清晰，易扩展为完整企业站
- 纯前端实现，无需 JavaScript

使用方法：
1. 将三个文件放在同一目录。
2. 在项目目录下创建 `images` 文件夹，放入：
   - banner.jpg
   - product1.jpg
   - product2.jpg
   - product3.jpg
3. 打开 index.html 即可在浏览器中查看效果。
