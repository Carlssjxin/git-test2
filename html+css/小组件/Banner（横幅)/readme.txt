# Banner 横幅组件（HTML + CSS）

本示例展示一个网页顶部的横幅（Banner），
包含一张大背景图和居中显示的标题文字。

文件说明：
- index.html ：HTML 页面结构
- style.css  ：CSS 样式文件
- README.txt ：说明文件

功能特性：
1. 背景图自动覆盖整个横幅区域（使用 background-size: cover）。
2. 标题文字居中显示（Flex 布局实现水平 + 垂直居中）。
3. 文字带有阴影（text-shadow）提升对比度和视觉层次感。
4. 横幅高度可自由调整（默认 400px）。

使用方法：
1. 将三个文件放在同一文件夹。
2. 替换 style.css 中的背景图片路径：`path/to/your-image.jpg`。
3. 打开 index.html 即可在浏览器中查看效果。
