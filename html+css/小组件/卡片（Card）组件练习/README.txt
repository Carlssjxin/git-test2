# 卡片组件练习（HTML + CSS）

本项目展示了一个简单的“卡片（Card）组件”结构。

文件说明：
- index.html ：HTML 页面结构
- style.css  ：样式文件
- README.txt ：说明文件

功能特性：
1. 卡片包含 图片 + 标题 + 简介 + 按钮。
2. 卡片宽度固定（320px），整体在页面中水平垂直居中。
3. 卡片带圆角与阴影效果。
4. 按钮在 hover（悬停）时会变色并加深阴影。

使用方法：
1. 将三个文件放在同一目录下。
2. 替换 index.html 中的图片路径为你本地图片路径。
3. 打开 index.html，在浏览器中查看效果。
