# 导航栏（Navbar）组件练习（HTML + CSS）

本示例展示一个带 Logo 与菜单项的导航栏组件，使用 Flex 实现水平居中与布局。

文件说明：
- index.html ：HTML 页面结构
- style.css  ：样式文件
- README.txt ：说明文件

功能特性：
1. 导航栏包含一个左侧 Logo 和右侧导航菜单。
2. 使用 Flex 布局让内容水平分布、垂直居中。
3. 鼠标悬停菜单项时文字变蓝并出现下划线。
4. 导航栏带轻微阴影与圆角感，视觉简洁清晰。
5. 设置 position: sticky；使导航栏在页面滚动时固定顶部。

使用方法：
1. 将三个文件放在同一文件夹中。
2. 打开 index.html 即可在浏览器中查看效果。
