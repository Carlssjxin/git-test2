# 页脚（Footer）组件练习（HTML + CSS）

本示例展示一个固定在底部的页脚，包含版权与联系方式。

文件说明：
- index.html ：HTML 页面结构
- style.css  ：CSS 样式文件
- README.txt ：项目说明

功能特性：
1. 页脚固定在浏览器底部，不随滚动而移动。
2. 背景为浅灰色，内容水平居中。
3. 包含两行信息：版权声明和联系方式（邮箱链接）。
4. 链接在 hover 时颜色加深，带平滑动画。
5. 页脚上方添加轻微阴影，区分主体与底部。

使用方法：
1. 将三个文件放在同一文件夹中。
2. 打开 index.html 即可在浏览器中查看效果。
3. 可将邮箱替换为你自己的联系方式。
